Object and surface labels

Display on-screen labels for objects and planetary surface features to make it easier to find or identify them. Label display can be toggled with AltF9 and configured via the Options dialog (F6). Labels are available for the following object types:

  • Markers for solar system bodies
  • Vessel markers
  • Surface base markers
  • Markers for navigation radio transmitter locations
  • Markers for planetary surface features
