Script functions

This section only defines Orbiter-specific extensions to the Lua interpreter. For general Lua functions, please consult a Lua manual.

The script functions available for accessing Orbiter state parameters are arranged in libraries according to function groups. In Lua, function libraries are grouped in tables. Functions are accessed by table name and function name, separated by a dot, e.g.

oapi.get_simtime()

Currently, five function lists are available: proc for process control functions, oapi for general interaction with the Orbiter API, vessel for obtaining vessel handles and instances, vec for doing mathematics on vectors and bit for managing bitfields (which are not available in Lua 5.1)

In addition to the function lists, there are also class methods available for four classes: vessel methods for interaction with a specific vessel, annotation methods for managing onscreen annotations, mfd methods for interaction with MFDs, and sketchpad for managing sketchpad objects.

Class methods act upon a given class instance, and use a colon syntax, such as:

v = vessel.get_focusinterface() -- obtain vessel instance
alt = v:get_altitude() -- invoke class method
