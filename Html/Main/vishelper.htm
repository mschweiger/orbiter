Visual helpers

Orbiter has the ability to display a number of visual cues to provide additional data to the user during the simulation. These include:

  • Planetarium mode
    Projects different coordinate grids onto the celestial sphere, displays constellations and provides markers for deep-space objects.
  • Object and surface labels
    Identify solar system bodies, vessels and planetary surface features.
  • Force vectors
    Shows the various forces acting on each spacecraft in the simulation.
  • Coordinate axes
    Shows the local reference frame for various simulation objects.

The visual helper display can be turned on and off and configured in the Options dialog (F6, or select Options from the main menu).
